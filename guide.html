User Guide

This guide will help you understand how to prepare your CSV, upload it, run predictions, and download results successfully.

1. Prepare Your CSV File

Your CSV must include the following columns:

  • Projects_Handled
  • Overtime_Hours
  • Sick_Days
  • Training_Hours
  • Employee_Satisfaction_Score
  • Work_Hours_Per_Week
  • Monthly_Salary
  • (Optional) Employee_ID
  • (Optional) Department

Make sure the CSV is clean and uses comma separators.

2. Upload Your File

Go to the Run Analysis page and upload the CSV using the upload box.

  • Click "Choose File" or the upload zone
  • Select your .csv file
  • Click Run Prediction

3. How the System Processes Your Data

The system automatically computes additional features:

  • Attendance_Rate = (1 - Sick_Days / 260) × 100
  • Salary_Per_Hour = Monthly_Salary / Work_Hours_Per_Week

Then the XGBoost model predicts one of the following performance levels:

  • Low
  • Below Average
  • Average
  • Above Average
  • High

4. Download Your Results

After the model finishes processing, a download button will appear. Click it to receive your:

  • Processed CSV
  • With predictions in Predicted_Performance

5. Tips for Best Accuracy

  • Avoid missing values
  • Ensure numeric fields contain numbers only
  • Use consistent units (hours, RM, etc.)
  • More employees = better insights
